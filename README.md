# Baiak-Zika Death Monitor

Este projeto é um monitorador de mortes do servidor Baiak-Zika. Ele realiza o scraping das últimas mortes do site e disponibiliza uma API para consulta, podendo ser integrado para envio de notificações via WhatsApp.

## Objetivo
Monitorar mortes de jogadores e monstros no Baiak-Zika e facilitar o envio dessas informações para o WhatsApp.

---
Projeto simples, feito em TypeScript, Express e com scraping automatizado.
